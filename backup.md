# Sistema de Backup

## Como garantir a segurança dos seus dados:

1. **Exporte regularmente:** Use a função de exportação
2. **Armazene externamente:** Salve os arquivos .json em drive externo
3. **Não limpe o cache:** Os dados ficam no navegador

## Em caso de problemas:
- Contate o suporte técnico do hospital
- Use a função de importação para restaurar backups
